Evolution Simulations

An exploration into how different statistical distributions explain genetic success and mutation.

Description of Project

This project is intended to be a light-hearted simulation of single-celled organisms evolving overtime. We will start off with a simple situation, where organisms compete with one another based on a fitness measure for space, ultimately resulting in one family tree of organisms dominating over a territory. Later examples will include different terrains, where only the most fit organisms will be able to expand their colonies into these regions. All simulations will have controls so that the user can modify initial fitness measures, and even the probability distributions used for organism mutation rates and death rates.

Simple Evolution

The properties of this simulation are the following. At each time step:

  1. Each organism can die with an equal probability.
  2. Given an organism has not died, it can reproduce, generating a uniform number of child organisms (up to 3) with the same color (representing that cell being a same member of the family).
  3. If an organism extends into another colony, then there will be a coin toss to determine which colony wins.

Genetic Drift

The properties of this simulation are the following. At each time step:

  1. Each organism can die with a probability corresponding to its fitness.
  2. Given an organism has not died, it can reproduce, generating a uniform number of child organisms (up to 3) with the same color (representing that cell being a same member of the family). Note that children here have a random mutation changing their fitness. To simulate genetic drift, new fitness will be based on a uniform distribution centered around the old fitness.
  3. If an organism extends into another colony, then there will be a coin toss with a probability proportional to the attacking cell's fitness to determine which colony wins.
